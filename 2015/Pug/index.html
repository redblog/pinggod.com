逻辑语句

Pug 提供了条件、分支、循环、遍历四种逻辑语句，这四种语句继承自 JavaScript，只是语法上有些差异:

  • 条件语句：if … else if … else
  • 分支语句：case … when … default
  • 循环语句：while
  • 遍历数组：each $elem in [elem…]
  • 遍历对象：each $key, $value in {key: value}
